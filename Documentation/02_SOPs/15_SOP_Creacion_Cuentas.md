# SOP 15: Creación de Cuentas y Perfiles

**Objetivo:** Estandarizar el alta de usuarios en los sistemas de la empresa.
**Responsable:** IT / Soporte.

## 1. Correo Corporativo (Google Workspace / Zoho)
*   **Formato:** `nombre.apellido@freeinterpreters.com`
*   **Password Inicial:** `Free2024!` (Usuario debe cambiarla al primer login).
*   **Grupos:** Añadir a `interpreters@`, `all@`.

## 2. Plataforma de Interpretación
*   Crear usuario con el ID asignado por RRHH.
*   Asignar "Skillset" o idiomas correspondientes (ej. Spanish Medical).
*   Configurar DID (número de entrada) si aplica.

## 3. CRM / Base de Datos
*   Ingresar datos completos: Nombre, ID, Email, Teléfono, País, Banco.
*   Subir documentos del expediente digital.

## 4. Desactivación
*   Al recibir notificación de baja, **desactivar** (no borrar) las cuentas inmediatamente para preservar el historial pero impedir el acceso.
