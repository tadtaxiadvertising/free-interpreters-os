# SOP 07: Emergencias Operativas

**Objetivo:** Establecer el protocolo de actuación ante situaciones imprevistas que afecten la continuidad del servicio.
**Responsable:** Todos los Intérpretes y Staff.

## Tipos de Emergencias
1.  **Corte de Energía Eléctrica.**
2.  **Corte de Internet (ISP).**
3.  **Emergencia Médica Súbita.**
4.  **Desastres Naturales.**

## Protocolo de Acción

### 1. Durante una Llamada Activa
*   **Si es posible:** Informar brevemente a las partes: *"Disculpen, tengo una emergencia técnica/personal, debo transferir la llamada."*
*   Transferir la llamada inmediatamente a la cola general o a otro intérprete.
*   Desconectarse.

### 2. Notificación
*   En cuanto sea seguro y posible, notificar al Supervisor vía WhatsApp o SMS (si no hay datos).
*   Mensaje: `EMERGENCIA - ID [ID] - [Motivo] - [Tiempo estimado de retorno]`.

### 3. Recuperación
*   **Corte de Luz/Internet:**
    *   Contactar al proveedor de servicios para obtener un número de reporte/ticket.
    *   Este número de reporte servirá como justificante para evitar penalidades por abandono de turno.
*   **Retorno:** Notificar cuando el servicio se restablezca y se esté listo para retomar el turno.

## Plan de Contingencia (Recomendado)
*   Tener un UPS (Sistema de Alimentación Ininterrumpida) para mantener el módem y PC encendidos unos minutos y cerrar procesos correctamente.
*   Tener una segunda opción de internet (Datos móviles / Hotspot) **SOLO para comunicar la emergencia**, no para interpretar (salvo autorización expresa).
