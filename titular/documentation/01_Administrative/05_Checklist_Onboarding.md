## Fase 1: Pre-Ingreso
- [ ] **Entrevista y Selección:**
    - [ ] Entrevista inicial aprobada.
    - [ ] Prueba de idioma (Roleplay) aprobada.
    - [ ] Validación de requisitos técnicos (Speedtest, Hardware).
- [ ] **Oferta y Contratación:**
    - [ ] Envío de Oferta Laboral.
    - [ ] Aceptación de Oferta.
    - [ ] Envío de Contrato Freelance.
    - [ ] Recepción de Contrato firmado.

## Fase 2: Documentación
- [ ] **Recolección de Documentos:**
    - [ ] Documento de Identidad (DNI/Pasaporte).
    - [ ] CV actualizado.
    - [ ] Certificados de idiomas (si aplica).
    - [ ] Antecedentes penales (si aplica).
    - [ ] Información bancaria/pagos.
    - [ ] Foto profesional para perfil.
- [ ] **Firma de Políticas:**
    - [ ] Código de Conducta firmado.
    - [ ] Acuerdo de Confidencialidad (NDA) firmado.
    - [ ] Reglamento Interno aceptado.

## Fase 3: Configuración Técnica
- [ ] **Accesos y Cuentas:**
    - [ ] Creación de correo corporativo (si aplica).
    - [ ] Alta en plataforma de interpretación.
    - [ ] Acceso a CRM/Portal de Intérpretes.
    - [ ] Inclusión en grupos de WhatsApp/Slack.
- [ ] **Pruebas de Sistema:**
    - [ ] Verificación final de audio y micrófono.
    - [ ] Instalación de software requerido.

## Fase 4: Capacitación (Training)
- [ ] **Inducción Corporativa:**
    - [ ] Bienvenida y Cultura de la empresa.
    - [ ] Revisión de SOPs y Manuales.
- [ ] **Entrenamiento Operativo:**
    - [ ] Uso de la plataforma de llamadas.
    - [ ] Protocolos de saludo y despedida.
    - [ ] Manejo de situaciones difíciles.
    - [ ] Glosarios básicos.

## Fase 5: Go Live
- [ ] **Primer Turno:**
    - [ ] Acompañamiento en primera conexión.
    - [ ] Primera llamada monitoreada.
    - [ ] Feedback inicial.
