# SOP 02: Onboarding Técnico

**Objetivo:** Asegurar que el intérprete tenga todos los accesos y configuraciones técnicas listas para operar sin inconvenientes.
**Responsable:** Soporte Técnico / IT.
**Herramientas:** TeamViewer/AnyDesk (opcional), Plataforma de Llamadas, Speedtest.

## Pasos del Proceso

### 1. Verificación de Hardware
*   Confirmar procesador (mínimo i3/Ryzen 3) y RAM (mínimo 8GB).
*   Verificar uso de conexión cableada (Ethernet). **Wi-Fi prohibido.**
*   Probar headset USB (audio y micrófono).

### 2. Creación de Cuentas
*   Crear correo corporativo `nombre.apellido@freeinterpreters.com` (si aplica).
*   Dar de alta en la plataforma de interpretación (VRI/OPI).
*   Configurar acceso al Portal de Intérpretes (CRM).

### 3. Configuración de Software
*   Instalar softphone o navegador requerido (Chrome/Edge actualizados).
*   Limpiar cookies y caché.
*   Desactivar actualizaciones automáticas de Windows durante horas laborales.
*   Configurar bloqueador de ruido (Krisp) si es necesario.

### 4. Prueba de Conexión
*   Ejecutar Speedtest.net:
    *   Ping < 50ms (ideal) / < 100ms (aceptable).
    *   Descarga > 20 Mbps.
    *   Carga > 5 Mbps.
    *   Jitter < 30ms.
    *   Packet Loss 0%.

### 5. Sesión de Prueba (Mock Call)
*   Realizar una llamada de prueba en la plataforma real.
*   Verificar claridad de audio y video.
*   Confirmar que el intérprete sabe usar los botones (Mute, Transfer, Hold).

## Checklist de Finalización
- [ ] Hardware validado.
- [ ] Internet validado.
- [ ] Credenciales entregadas y probadas.
- [ ] Intérprete sabe loguearse y desloguearse.
